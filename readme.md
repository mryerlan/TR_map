# TR MAP Version 2.0

This project makes a sequence of all transport nodes for each base station. 
Based on the mac address table and routing tables

List of supported equipment:
* Mini-Link TN
* Mini-Link CN210
* Mini-Link 6352
* Mini-Link 6000
* Optix RTN 900
* Optix RTN 300

List of required reports

SoEM:
* Ne inventory
* Ne ethernet

NCE
* Cable report
* NE report
* Self mac address report
